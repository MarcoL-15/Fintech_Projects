# P2P Lending Logistic regression

The 'Optimizing Automated Credit Issuance: P2P Republic' initiative tackles non-performing loans by proposing an algorithm-driven underwriting framework. The focus lies on peer-to-peer lending, where abundant data enables the development of an automated credit assessment system. Rather than relying on subjective and time-consuming manual evaluations, the project employs a predictive scoring model to enhance default detection while preserving portfolio integrity. 

Leveraging the Lending Club dataset (containing 390,000+ loans spanning 2007–2016), the problem was formulated as a binary classification task: predicting loan repayment status based on applicant attributes. Following data preprocessing and feature selection, logistic regression was used to identify which customer metrics most strongly influence repayment likelihood. The analysis revealed significant relationships between default risk and variables including loan size, borrower income, debt-to-income ratio, employment history, and identity verification. Validation tests confirmed variable stability, and statistical analysis pinpointed the most influential default predictors. The resulting model achieved an AUC of 0.71, demonstrating reliable discrimination between performing and non-performing loans. The study also highlighted the critical role of decision thresholds, revealing the inherent trade-off between loan approval rates and default minimization. Within banking, stricter thresholds are often justified given the severe impact of non-performing loans on profitability, even if this rejects some viable borrowers. The project illustrates how algorithmic underwriting can elevate credit decisions and mitigate defaults—a scalable approach applicable across fintech lenders, traditional banks, and other financial institutions offering credit products.
