# Fintech_Projects
A collection of Fintech projects developed during my MSc at Politecnico di Milano and independently. Showcasing my experience and interests in Finance and Digital Innovation, leveraging Python, R, and Solidity. Projects span topics such as portfolio optimisation, VaR modelling, time series analysis, supervised and unsupervised learning, and blockchain development using Python, R, and Solidity — each accompanied by a brief description of the objective, methodology, and key findings. This repository is actively growing, and as new projects are developed they will be added here.
